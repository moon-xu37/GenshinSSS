# 模式参数

| 模式参数 | 别名      | 说明                     |
| -------- | --------- | ------------------------ |
| EXPECT   | e, 期望   | 计算伤害的期望(默认)     |
| CRIT     | c, 暴击   | 计算一个必定暴击的伤害   |
| NOTCRIT  | n, 不暴击 | 计算一个必定不暴击的伤害 |
| MISS     | m, 缺失   | 不造成伤害               |
| SIM      | s, 模拟   | 生成随机数模拟一次伤害   |
